# 🐾 SNZ Shelter — сайт приюта для животных

Полноценный веб-сайт для приюта «Друг из приюта», написанный с нуля на Django.
Проект отображает животных, ведёт блог с полезными статьями, а также содержит расширенную админ-панель для сотрудников приюта.

Сайт работоспособен и задеплоен на сервер. Его можно найти по адресу [https://drugizpriyuta.ru/](https://drugizpriyuta.ru/)

# ✨ Основной функционал
## 👇 Публичная часть сайта

📄 Каталог животных с краткими карточками

🐶 Страница животного — полная информация (история, особенности, фото)

📝 Блог / статьи — полезные материалы о животных, уходе и пристройстве

🔍 Фильтрация животных по полу, возрасту, в случае собак - по росту

## 🔧 Админ-панель (Django Admin)

Расширенная административная зона для сотрудников приюта:

🩺 Детальная карточка животного (здоровье, история, заметки)

📚 База данных будущих и текущих владельцев

🏠 Управление вольерами (генерация карточек для вольеров — в процессе разработки)

✍️ Управление блогом и материалами

🪙 Финансовая отчетность

## 🛠️ Технологии

Backend:

Python 3

Django (с чистыми Django Templates)

Frontend:

HTML5

CSS (без фреймворков)

База данных:

PostgreSQL

Все упаковано в контейнеры с использованием Docker

Проект создан как рабочий сайт реального приюта.
